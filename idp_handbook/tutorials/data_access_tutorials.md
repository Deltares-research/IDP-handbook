# Data access tutorials

In this section we provide tutorials on accessing, subsetting and visualizing different types of cloud-optimized datasets: GeoTIFF, zarr and Parquet. Below is the list of the currently available tutorials:

<div class="full-width">

:::{list-table}
:header-rows: 1

*   - Tutorial
    - Dataset format
    - Dataset name
    - Data catalog
    - Description

*   - [GeoTIFF - sea level rise](./IDP-workbench/tutorials/co-data/access_stac_geotiff.ipynb)
    - GeoTIFF
    - Sea level rise projections IPCC AR6
    - CoCliCo STAC
    - Exploring STAC catalog contents, selecting and retrieving data layers in GeoTIFF format, visualizing data on an interactive plot, calculating aggregate statistics.

*   - [zarr - weather & climate](./IDP-workbench/tutorials/co-data/access_stac_zarr.ipynb)
    - zarr
    - ERA5 reanalysis
    - DestinE Platform
    - Accessing the cloud-optimized ERA5 single-levels dataset, deriving variables, subsetting by area and visualizing data along time and spatial dimensions.

*   - [GeoParquet - critical infrastructure](./IDP-workbench/tutorials/co-data/access_stac_parquet.ipynb)
    - Parquet
    - EU-wide critical infrastructure
    - CoCliCo STAC
    - Accessing vector data in Parquet format, subsetting by area of interest and plotting objects on an interactive map.
